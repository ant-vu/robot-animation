# **Robot Animation**

## **Installation**

To install Robot Animation, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/ant-vu/robot-animation.git
```

2. Install dependencies:

```bash
npm install
```

3. Start the project:

```bash
npm run dev
```
